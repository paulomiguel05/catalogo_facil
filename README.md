   O Software inclui:

// Aplicação web local com acessível pela rede.

// Sistema de gestão comercial para pequenas lojas.

// Sistema de catálogo digital + Personalização de UX

// Software de gestão de produto, clientes e vendas.

   O Foco deste software é servir de formar acessível um a gestão de estoque para pequenas lojas e pessoas que trabalham com vendas com contato direto a seus clientes, tornando o que normalmente vemos sempre sendo feito fisicamente um pouco mais tecnológico.



## Com o uso de Catálogo Fácil a 'loja' terá:



* Controle de seu estoque de produtos:
   - Cadastrar / Editar / Excluir produtos (\* Imagens \* Preço de Custo \* Preço de venda \* Descrição \* Categorização \*)
   - Consultar estoque por "Busca" e filtro de ordenação
   - Gestão de categorias de produtos.

* Registro de vendas: 
   - Venda vinculada com vários itens.
   - Status de pagamento de venda (pago ou pendente).
   - Lista valor de lucro e custo, conectado ao cliente.
   - Atualiza estoque de item vendido.
   - Possível cancelar venda.
   - Consultas de faturamento e lucro por data e por categoria.
   - Cobrar caso status pendente com apenas 2 clicks.

* Lista de Clientes:
   - Cadastrar / Editar / Excluir clientes ( \* Nome \* Numero \* CPF \* Endereço \* )
   - Buscar clientes por nome, telefone e CPF.
   - Visualizar histórico de compras.
   - Status de pendências e cobrar com 2 clicks (total de venda e venda unica)

* Catálogos digitais
   - Criar / Customizar / Editar / Excluir catalogos (\* Logo \* Redes sociais da Loja \* Preço de venda \* Descrição de produto \* Quantidade \* Categorias ativas \* Carrinho de compra para seu cliente fazer pedido em seu whatsapp com poucos cliques ( com boa UX! ) 
   - Personalição de Cores do catalogo.
   - Baixar / Exportar Catalogo com todas as animações, parecendo um site! Obs: Abrir no navegador, catalogo não atualiza conforme o estoque, por isso sempre avisar que ele é do dia mencionado nem seu nome de arquivo.

* Acesso local pelo celular:
- Abrindo o sistema no seu computador, ele fica disponivel para ser acessador por qualquer dispositivo em sua rede, de forma oculta e segura!

Obs: O catalogo só funcionara em interpretadores de HTML, CSS e JavaScript (Navegadores).
